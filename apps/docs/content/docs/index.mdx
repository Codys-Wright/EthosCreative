# Ethos Creative Monorepo

Welcome to the Ethos Creative monorepo documentation. This repository is built using Turborepo and contains multiple packages and applications.

## Structure

```bash
ethos-creative/
├── apps/
│   ├── docs/          # Documentation site
│   ├── playground/    # Component playground
│   ├── web/          # Main web application
│   └── my-artist-type/ # Artist type application
└── packages/
    ├── ui/           # Shared UI components
    ├── auth/         # Authentication package
    ├── database/     # Database utilities
    ├── email/        # Email templates and utilities
    ├── eslint-config/ # Shared ESLint configuration
    └── typescript-config/ # Shared TypeScript configuration
```

## Getting Started

1. Install dependencies:
```bash
pnpm install
```

2. Start development:
```bash
pnpm dev
```

## Packages

### UI (@repo/ui)
A comprehensive UI component library built with:
- React 19
- Radix UI primitives
- Tailwind CSS
- shadcn/ui styling

To use components:
```tsx
import { Button, Badge } from "@repo/ui";
```

### Auth (@repo/auth)
Authentication utilities and components.

### Database (@repo/database)
Database utilities using:
- Drizzle ORM
- PostgreSQL

### Email (@repo/email)
Email templates and sending utilities using:
- React Email
- Resend

## Development

### Adding Dependencies
To add a dependency to a workspace:
```bash
pnpm add <package> --filter <workspace>
```

Example:
```bash
pnpm add dayjs --filter @repo/ui
```

### Workspace Dependencies
Internal dependencies should use the `workspace:*` protocol:
```json
{
  "dependencies": {
    "@repo/ui": "workspace:*"
  }
}
```
